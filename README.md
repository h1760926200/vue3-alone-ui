# vue3-alone-ui

vue3-alone-ui是一款个人组件库，使用vue3框架，对Element UI一些组件进行二次封装以及原生组件封装。

## Install

```
npm i vue3-alone-ui
国内网站请访问：http://gardon.top/#/home
项目中目前只允许全局使用 还未开发按需导入
import aloneUi from 'vue3-alone-ui'
import 'vue3-alone-ui/dist/vue3-alone-ui.css'
app.use(aloneUi)
```
